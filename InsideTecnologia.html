Inside Tecnologia

Inside Tecnologia is a company that develops custom interactive educational software to help other companies capacitate their employees by using desktop, VR, and AR experiences.

I'm currently working at Inside Tecnologia since September 2021 as a full-time Junior Unity Developer and took part in various projects shipping in different formats and platforms, such as Android VR apps on Meta Quest's Applab, Windows executables, WebGL, and Android apps.

More information about the company and its projects is available at: https://insidetecnologia.com/

Here is some of the work I’ve done on the company’s projects:

Inside Showroom

This project is meant to be a showcase of what the company can do for its clients, containing various types of VR interactions, systems, and activities, as well as a fully synchronized multiplayer mode. It was made for Meta Quest devices using Unity’s XR interaction toolkit, along with some features from Meta’s interaction SDK. And for the multiplayer, Photon Pun 2, Photon Voice, and Salsa.

Jornada Espiral

Made for Espiral Engenharia, a company specializing in scaffolds, this project is meant to evaluate and capacitate the workers in some of the core activities for an efficient and safe work environment, such as correctly identifying the scaffold pieces by their name and hand gesture, assembling a scaffold project based on the blueprint, and simulating accident scenarios and how to prevent them. The project is built for Meta Quest devices and was made using Meta’s interaction SDK.

Scaffold assembly system

Samarco PCRCs

Made for Samarco Mineração, an industrial mining company, this project promotes safety in the work environment for its employees by presenting several interactive scenarios, each representing a PCRC norm the workers must follow to ensure a safe work space. It was made for Meta Quest devices using Unity’s XR interaction toolkit.

Objective-based dynamic pathfinder system and shader
